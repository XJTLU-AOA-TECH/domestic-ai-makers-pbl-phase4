# Lab 03 — Short Press and Long Press

## Goal

Recognize different press actions.

- Short press: switch mode
- Long press: reset mode

## Wiring

Use the same button wiring as Lab 01.

## Expected result

Serial Monitor shows whether the press was short or long.

```text
Short press: switch mode
Long press: reset
```

## Challenge

Add double-click detection.
---

## 中文说明

本实验识别短按和长按。短按可以切换模式，长按可以重置系统，这是很多智能硬件常见的交互方式。
