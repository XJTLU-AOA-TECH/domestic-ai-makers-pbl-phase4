# Difficulty Levels

## Purpose

Not every team needs to complete every advanced feature. Use these levels to set realistic goals.

## Day 1

| Level | Target |
| --- | --- |
| Basic | Screen/menu layout is designed or represented |
| Standard | Button or touch sensor changes menu state |
| Advanced | Real touchscreen menu control works |
| Extension | AI improves screen layout and interaction flow |

## Day 2

| Level | Target | Lab |
| --- | --- | --- |
| Basic | LED/light output works | `01-led-feedback` |
| Standard | Bluetooth light control works | `05-bluetooth-light-control` |
| Advanced | Wi-Fi light control works | `06-wifi-light-control` |
| Extension | AI-designed command protocol is documented | Student design |

## Day 3

| Level | Target | Lab |
| --- | --- | --- |
| Basic | Motor and servo work separately | `05-car-motion-program` |
| Standard | Car actions work with wheels lifted | `05-car-motion-program` |
| Advanced | Speed and servo direction are adjustable | Student design |
| Extension | Touchscreen or Serial control panel sends car commands | Student design |

## Day 4

| Level | Target | Lab |
| --- | --- | --- |
| Basic | Car action functions work | `01-single-motor-test`, `02-dual-motor-actions` |
| Standard | Bluetooth remote car works | `03-bluetooth-car-control` |
| Advanced | Wi-Fi web car control works | `04-wifi-web-car-control` |
| Extension | AI-assisted motor decision rules are added | Student design |

## Day 5

| Level | Target | Lab |
| --- | --- | --- |
| Basic | Rule-based decision prints correct actions | `01-rule-based-decision` |
| Standard | Manual/Auto mode switch works | `02-manual-auto-mode` |
| Advanced | Sensor input controls pathfinding behavior | `03-sensor-triggered-control`, `05-intelligent-pathfinding-project` |
| Extension | AI tag/label or voice command controls car behavior | `04-edge-ai-smart-car-project`, student design |

## Teaching recommendation

For beginners, require Basic level and encourage Standard level. Treat Advanced and Extension levels as challenges for faster teams.

For Day 4 and Day 5, Bluetooth, Wi-Fi, motors, and batteries can introduce many classroom variables. A safe stop command and low-speed test are required before car demos.

## 中文说明

本文件用于降低课程难度压力。不是所有学生都必须完成最高难度。对于初学者，每一天完成 Basic level 就算达成基础目标；Standard level 是大部分小组的正常目标；Advanced 和 Extension 适合作为进阶挑战。

Day 5 的主线是智能寻路小车，语音控制小车可以作为拓展方向。
