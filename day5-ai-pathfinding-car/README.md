# Day 5 — AI Pathfinding Car

## Goal

Students build an intelligent pathfinding car by mapping AI tags, sensor states, or classification labels to motor and servo responses.

By the end of Day 5, students should be able to:

1. Explain how AI labels or sensor states can control hardware actions.
2. Switch between manual mode and automatic mode.
3. Use sensor input to trigger car actions.
4. Design pathfinding rules based on distance or labels.
5. Build an intelligent pathfinding car as the final project.

## Labs

| Lab | Description |
| --- | --- |
| `01-rule-based-decision` | Map sensor states to action decisions |
| `02-manual-auto-mode` | Switch between manual wireless control and automatic control |
| `03-sensor-triggered-control` | Use distance or other sensor input to control car behavior |
| `04-edge-ai-smart-car-project` | Build an Edge AI-style smart car control prototype |
| `05-intelligent-pathfinding-project` | Build the final intelligent pathfinding car |

## Key concepts

- AI tag / AI label: a classification result such as `left`, `right`, `stop`, or `target`.
- Decision mapping: convert labels or sensor states into motor/servo actions.
- Pathfinding: choose a safe or target-oriented movement path.
- Manual mode: user controls directly through Bluetooth, Wi-Fi, or touchscreen.
- Auto mode: device chooses actions based on sensor/perception input.

## Project levels

| Level | Requirement | Who should do it |
| --- | --- | --- |
| Basic | Manual control works and car can stop safely | Everyone |
| Standard | Manual/Auto mode switch works | Most teams |
| Advanced | Sensor input automatically changes car actions | Faster teams |
| Extension | Add AI labels, visual/audio input, or voice-controlled car | Optional challenge |

## Suggested classroom flow

1. 20 min — AI tag / label to hardware response concept.
2. 30 min — Rule decision and manual/auto mode review.
3. 40 min — Sensor-triggered pathfinding.
4. 55 min — Intelligent pathfinding car final project.
5. 25 min — Demo, reflection, and improvement ideas.
