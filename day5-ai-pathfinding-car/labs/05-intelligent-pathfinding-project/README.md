# Lab 05 — Intelligent Pathfinding Project

## Goal

Build the final intelligent pathfinding car project.

The car should use sensor states, AI tags, or classification labels to choose motor and servo actions.

## Project idea

Input examples:

- Distance sensor: `close`, `clear`
- AI tag / label: `left`, `right`, `stop`, `target`
- Optional sound or voice label: `go`, `stop`

Output examples:

- `forward()`
- `turnLeft()`
- `turnRight()`
- `stopCar()`
- servo scan or direction adjustment

## Requirements

Your project should:

1. Include a safe stop action.
2. Use at least one sensor or AI label as input.
3. Map input states to motor/servo responses.
4. Demonstrate simple pathfinding or obstacle avoidance.
5. Explain the decision rules.

## Extension

Add voice-controlled car behavior from Phase 3 Day 6 audio recognition.
