# Lab 03 — Sensor-Triggered Control

## Goal

Use sensor input to automatically control car behavior.

Example:

```text
Distance close → stop
Distance medium → slow forward
Distance far → forward
```

## Expected result

The car changes action based on sensor readings.

## Safety

Keep the wheels lifted during the first test. Start with low speed.

## Challenge

Add a sound event or visual state as another trigger.
---

## 中文说明

本实验使用传感器输入自动控制小车动作。学生将理解“感知结果如何变成执行动作”。
