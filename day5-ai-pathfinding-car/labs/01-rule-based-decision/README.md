# Lab 01 — Rule-Based Decision

## Goal

Map sensor states to car actions using simple rules.

## Example

```text
If distance is close → STOP
If distance is safe → FORWARD
If sound event detected → STOP
```

## Expected result

Serial Monitor prints the sensor state and the selected action.

## Challenge

Add one more input, such as sound level or touch mode.
---

## 中文说明

本实验用规则决策把传感器状态转换成小车动作。规则决策是理解 AI 决策之前最基础、最清晰的方法。
