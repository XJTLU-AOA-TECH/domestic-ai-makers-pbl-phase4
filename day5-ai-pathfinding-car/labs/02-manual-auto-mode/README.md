# Lab 02 — Manual and Auto Mode

## Goal

Switch between manual mode and automatic mode.

- Manual mode: user sends commands.
- Auto mode: device decides action from sensor input.

## Expected result

Serial Monitor accepts commands:

```text
M → manual mode
A → auto mode
F/B/L/R/S → manual movement commands
```

## Challenge

Use Bluetooth or Wi-Fi to send the same mode commands.
---

## 中文说明

本实验实现手动模式和自动模式切换。手动模式由用户控制，自动模式由设备根据传感器输入决定动作。
