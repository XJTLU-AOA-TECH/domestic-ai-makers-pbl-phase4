# Lab 05 — Car Motion Program

## Goal

Build a complete car motion program using DC motors and an optional servo.

## Required actions

```cpp
forward();
backward();
turnLeft();
turnRight();
stopCar();
```

## Optional servo actions

```cpp
lookLeft();
lookCenter();
lookRight();
```

## Expected result

With the wheels lifted, the car performs each movement action safely.

## Challenge

Add a speed variable and test low, medium, and high speed.
