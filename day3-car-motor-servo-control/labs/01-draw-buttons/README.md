# Lab 01 — Draw Buttons

## Goal

Draw simple buttons on a touchscreen or display.

## Hardware

Use the touchscreen/display module available in your kit. Different modules require different libraries, so this lab provides UI logic that can be adapted.

## Expected result

The screen shows:

```text
[START] [STOP]
```

## Challenge

Add a third button: `MODE`.
---

## 中文说明

本实验绘制简单按钮界面。不同触摸屏模块的库可能不同，重点是理解按钮区域和界面布局。
