# Lab 03 — Speed Slider

## Goal

Convert touch position into a speed value.

## Expected result

Touching different positions on the slider changes the speed value:

```text
Speed: 20
Speed: 65
Speed: 100
```

## Challenge

Use the speed value to control LED blink speed or motor PWM in a later lesson.
---

## 中文说明

本实验用触摸位置控制数值。滑块是触摸屏界面中常见的交互组件，可以用来设置速度、亮度或音量。
