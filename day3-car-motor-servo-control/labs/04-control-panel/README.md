# Lab 04 — Control Panel

## Goal

Build a touchscreen control panel prototype.

The interface can include:

```text
[Forward] [Stop] [Back]
[Left]    [Mode] [Right]
Speed: 60%
```

## Requirements

Your prototype should:

1. Show at least four buttons.
2. Detect which button is touched.
3. Maintain a current mode or speed value.
4. Print the command in Serial Monitor.

## Challenge

Use this panel later to control the smart car in Lesson 4.
