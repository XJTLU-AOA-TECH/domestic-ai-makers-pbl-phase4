# Lab 02 — Touch Button Detection

## Goal

Detect which button area is touched.

## Expected result

When a touch coordinate is inside a button rectangle, Serial Monitor prints the button name.

```text
Touched: START
Touched: STOP
```

## Challenge

Add a `MODE` button and update the detection logic.
---

## 中文说明

本实验学习按钮区域判断。程序通过触摸点坐标判断用户点击了哪个按钮。
