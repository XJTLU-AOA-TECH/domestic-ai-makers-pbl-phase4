# Lab 01 — LED Feedback

## Goal

Use LED patterns to show different device states.

## Wiring

- LED → GPIO 2 through resistor
- LED GND → GND

## Expected result

The LED uses different patterns:

- Slow blink: waiting
- Fast blink: active
- Solid on: confirmed

## Challenge

Add a button or touch input to switch between states.
---

## 中文说明

本实验使用 LED 闪烁方式表达设备状态。不同闪烁频率代表不同反馈信息。
