# Lab 05 — Bluetooth Light Control

## Goal

Control an LED or light module through Bluetooth commands.

## Commands

```text
ON    -> turn light on
OFF   -> turn light off
BLINK -> blink mode
RED   -> red mode if using RGB/LED strip
BLUE  -> blue mode if using RGB/LED strip
```

## Expected result

A phone or computer sends Bluetooth commands and the light changes state.

## AI challenge

Ask AI to design a clear command format and response messages for the light controller.
