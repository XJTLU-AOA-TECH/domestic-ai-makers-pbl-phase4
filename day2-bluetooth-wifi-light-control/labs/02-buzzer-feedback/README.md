# Lab 02 — Buzzer Feedback

## Goal

Use a buzzer to give sound feedback.

## Wiring

- Buzzer signal → GPIO 4
- Buzzer GND → GND

## Expected result

The buzzer plays short tones to confirm actions.

## Challenge

Create different tones for success, warning, and error.
---

## 中文说明

本实验使用蜂鸣器提供声音反馈。不同声音可以表示成功、警告或错误。
