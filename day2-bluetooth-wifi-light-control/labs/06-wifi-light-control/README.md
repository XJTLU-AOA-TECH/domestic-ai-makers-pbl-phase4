# Lab 06 — Wi-Fi Light Control

## Goal

Control an LED or light module from a phone browser through Wi-Fi.

## Setup

1. Replace `YOUR_WIFI_SSID` and `YOUR_WIFI_PASSWORD` locally.
2. Upload the code.
3. Open Serial Monitor to find the board IP address.
4. Open the IP address in a browser on the same network.

## Important

Do not commit real Wi-Fi passwords to GitHub.

## Expected result

The web page shows buttons for ON, OFF, and BLINK.
