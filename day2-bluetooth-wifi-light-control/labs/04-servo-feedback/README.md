# Lab 04 — Servo Feedback

## Goal

Use servo movement as physical feedback.

## Wiring

- Servo signal → GPIO 10
- Servo VCC → 5V recommended
- Servo GND → common GND

Use external power if the servo is unstable.

## Expected result

The servo moves between 0, 90, and 180 degrees.

## Challenge

Move the servo to a different angle for each interaction state.
---

## 中文说明

本实验使用舵机作为动作反馈。舵机可以转到指定角度，适合表示开关、指针或机械动作状态。
