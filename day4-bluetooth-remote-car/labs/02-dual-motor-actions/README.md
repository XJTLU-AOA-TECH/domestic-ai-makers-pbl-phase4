# Lab 02 — Dual Motor Actions

## Goal

Create basic car movement functions.

## Actions

```cpp
forward();
backward();
turnLeft();
turnRight();
stopCar();
```

## Expected result

With the wheels lifted, the car performs each action in sequence.

## Challenge

If the car turns the wrong way, fix the motor direction by swapping wires or changing the logic.
---

## 中文说明

本实验把左右两个电机封装成小车动作函数。后续蓝牙、Wi-Fi 和 AI 控制都会调用这些函数。
